ns_slider
=========

Slider (carusel like) web component based on Polymer

Usage
-----

Place your elements in component content

```html
	<ns_slider>
		<div>
			<img src="image1.jpg" />
			<div class="title">Some title 1</div>
		</div>
		<div>
			<img src="image2.jpg" />
			<div class="title">Some title 2</div>
		</div>
	</ns_slider>
```
